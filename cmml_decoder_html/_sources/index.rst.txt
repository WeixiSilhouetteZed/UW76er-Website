Introduction
------------

This is a wrapper class for the CrowdMark website to help students monitoring their assessments' marking processes.

Software Requirements
---------------------

This ``CmmlDecoder`` class requires ``Pandas``, ``seleniumrequest``, and ``json`` libraries to run. All of them can be installed using
``pip``.
